# Easy wins in React

1. <StricMode> Component
[Doc](https://react.dev/reference/react/StrictMode)

Wrap your entire App to get good practices check during development

```jsx
<StrictMode>
  <App />
</StrictMode>
```

2. <Suspense> Component
[Doc](https://react.dev/reference/react/Suspense)

Improve UX by displaying fallback when view is loading. It works great with [lazy loading](https://react.dev/reference/react/lazy)

```jsx
<Suspense fallback={<Loading />}>
  <SomeComponent />
</Suspense>
```

3. ErrorBoundary Coponent

[Doc](https://react.dev/reference/react/Component#catching-rendering-errors-with-an-error-boundary)
Wrap your app with ErrorBoundry, to render user friendly error message when app rendering crash
